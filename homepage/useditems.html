Here is a list of all the items I used while creating the website:


Basic elements

  1. margin = used to create space around elements
  2. width = used to set the width of an element
  3. height = used to set the height of an element
  4. padding = used to create space inside an element, between the element's content and its border
  5. color = set the color of the text inside an element
  6. background-color = used to set the background color of an element
  7. border = used to set the border around an element

Position elements

  1. static = the element is positioned according to the normal flow of the document
  2. relative = the element is positioned relative to its normal position
  3. absolute = the element is positioned relative to its nearest positioned ancestor
  4. fixed = the element is positioned relative to the browser window
  5. sticky = the element is positioned based on the user's scroll position
  6. inherit = used to make an element inherit the value of a property from its parent element

Entities

  1. & used as the starting point for HTML character entities
  2. " " used for defining strings, such as values in content, attribute selectors, and URL functions
  3. < > used to define opening and closing tags for elements
  4. & nbsp used as a space character that prevents a line break
  5. &euro used to represent the Euro sign
  6. & copy represents the copyright symbol
  7. & trade; used to insert the trademark symbol (™)
